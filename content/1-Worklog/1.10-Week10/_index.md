> **Duration:** 15/06/2026 - 21/06/2026 Implementing the Query Layer and Dashboard for the Data Lakehouse

---

### Weekly Objectives

- Complete the Analytics & Visualization Layer of the Customer Behavior Analytics system on AWS.
- Build a data querying environment using Amazon Athena integrated with the AWS Glue Data Catalog.
- Develop an interactive dashboard using Streamlit and integrate it with AWS analytics services.
- Deploy the dashboard on Amazon EC2, perform end-to-end system testing, and ensure public accessibility over the Internet.

---

### Work Activities

- Configure **Amazon Athena** to query datasets registered in the **AWS Glue Data Catalog**.
- Validate the datasets in the **Gold Layer** using SQL queries and evaluate query performance on **Apache Parquet** files.
- Develop a **Streamlit** dashboard to visualize analytical data generated by the Data Lakehouse.
- Integrate **AWS SDK for pandas (AWS Wrangler)** with the dashboard to enable direct querying of the Gold Layer through **Amazon Athena**.
- Design the dashboard interface with key performance indicators (KPIs) and analytical visualizations, including **Total Revenue**, **Total Orders**, **Total Customers**, **Average Order Value**, **Revenue Over Time**, **Revenue by Country**, **Device Summary**, **Payment Summary**, and **Traffic Source Summary**.
- Provision and configure an **Amazon EC2** instance, including setting up a **VPC**, **Public Subnet**, **Internet Gateway**, **Route Table**, and **Security Group** for application deployment.
- Install the Python runtime, required libraries, configure the **AWS CLI**, and set up AWS credentials so the dashboard can securely access AWS services.
- Deploy the Streamlit application on Amazon EC2, configure it to run as a background service, and verify dashboard accessibility through the EC2 Public IP address.
- Perform end-to-end testing of the data flow from the **Gold Layer → Amazon Athena → Streamlit Dashboard** to ensure accurate data visualization and stable system performance.

---

### Achievements

- Successfully completed the **Analytics & Visualization Layer** of the Customer Behavior Analytics system.
- Successfully established a data querying environment using **Amazon Athena** integrated with the **AWS Glue Data Catalog**.
- Developed a fully functional **Streamlit dashboard** featuring comprehensive KPI metrics and interactive analytical visualizations.
- Successfully deployed the dashboard on **Amazon EC2**, enabling users to access the application via its public IP address.
- Verified that the dashboard accurately retrieves data from the **Gold Layer** through **Amazon Athena**, ensuring consistency between the displayed analytics and the underlying datasets.
- Completed the analytics, visualization, and deployment phase of the project, establishing a solid foundation for end-to-end testing and evaluation of the Customer Behavior Analytics pipeline on AWS.
