> **Duration:** 11/05/2026 - 17/05/2026 Data Analytics and Business Intelligence (BI)

---

### Weekly Objectives

- Learn how to use Amazon QuickSight to build professional dashboards that support data-driven decision-making.
- Understand the different analytics services available on AWS and learn how to choose the most appropriate service for specific use cases.
- Apply analytics skills to cloud resource management by analyzing operational performance and cloud costs.
- Combine the knowledge and skills acquired in previous weeks to prepare high-quality datasets for data visualization.

---

### Learning Contents

#### 1. Data Visualization with Amazon QuickSight

- **Business Intelligence with Amazon QuickSight:** Practice connecting Amazon QuickSight to data sources prepared in previous weeks, such as Amazon S3, Amazon Athena, and Amazon RDS. Learn how to build interactive dashboards, perform advanced data analysis, and share reports with other users.
- **Data Analytics Services Overview:** Learn the differences between AWS analytics services such as Amazon EMR, Amazon Redshift, and Amazon Athena, and understand how to select the right service based on data size, workload, and business requirements.

#### 2. Advanced Analytics

- **Serverless Analytics with Amazon Athena:** Practice writing SQL queries to analyze large datasets stored directly in Amazon S3 without provisioning or managing any server infrastructure.
- **Advanced PostgreSQL on AWS (Part 1 & Part 2):** Explore advanced PostgreSQL administration and query optimization techniques. Strengthen advanced SQL skills and perform feature engineering directly within the database layer.

#### 3. Operational and Cost Analytics

- **Cost Visualization and Analytics:** Practice using visualization tools to analyze AWS Cost and Usage Reports (CUR), enabling better visibility into cloud spending.
- **Cost Data Analysis with AWS Glue and Amazon Athena:** Combine AWS Glue and Amazon Athena to transform and query complex cost datasets, enabling data-driven decisions for cloud resource optimization.
- **Cost and Usage Management:** Learn best practices and AWS tools for monitoring and managing cloud resource consumption efficiently.

#### 4. Performance Monitoring and Visualization

- **Advanced Monitoring with CloudWatch and Grafana:** Learn how to integrate Amazon CloudWatch metrics with Grafana to build real-time performance dashboards for cloud infrastructure and data pipelines.

#### 5. Resource Management and Analytics Access

- **Resource Organization with Tags and Resource Groups:** Learn how to organize AWS resources using tags, making it easier to manage projects and filter cost analytics across different environments.

---

### Key Takeaways

- **Understanding the value of data visualization:** Amazon QuickSight transforms raw data into interactive charts and dashboards, making insights easier to understand and communicate, especially to stakeholders without technical backgrounds.
- **Choosing the right analytics service:** Gain a clear understanding of the strengths and use cases of Amazon Athena, Amazon Redshift, and Amazon QuickSight, enabling you to select the most appropriate service for querying data, managing large-scale data warehouses, or visualizing analytical results.
- **Recognizing the importance of cost optimization in data analytics:** In addition to building effective analytics solutions, monitoring and optimizing cloud resource costs is essential for maintaining an efficient and cost-effective infrastructure.
- **Understanding the end-to-end analytics workflow on AWS:** Amazon S3, Amazon Athena, and Amazon QuickSight integrate seamlessly to form a complete analytics pipeline, from data storage and querying to visualization and business reporting.
