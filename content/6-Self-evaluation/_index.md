Throughout my internship at **Amazon Web Services Vietnam Co., Ltd.** from **20/04/2026 to 15/07/2026**, with a focus on **Data Engineering**, I developed the project "FinOps-Optimized Serverless Medallion Data Lakehouse Architecture for Customer Behavior Analytics on AWS." This project provided me with the opportunity to apply the knowledge I had learned to build a complete Data Lakehouse solution on AWS. The system was designed using a Serverless Medallion Data Lakehouse architecture, incorporating both Batch Processing and Streaming Processing for data ingestion, AWS Glue ETL for data transformation, Raw, Bronze, Silver, and Gold data layers for storage and organization, Amazon Athena for querying, and a Streamlit dashboard deployed on Amazon EC2 for data visualization.
